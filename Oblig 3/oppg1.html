Oppgave 1

Forklar med dine egne ord hva en funkjson er, og hvorfor vi lager funksjoner. Forklar også begrepene argument, parameter og returverdi.

En javascript funkjson er en del kode som er laget for å utføre en oppgave, som for eksempel legge sammen 2 verdier, sjekke om noe stemmer osv.

Vi lager funksjoner for at de skal utføre en oppgave i programmet, så ikke programmet blir rotete med all kode i en funksjon. De kan bli utført når noe ber den utfløre seg, kan være kommander som document.getElementById("btn").onclick /onfocus , Window.onload = funksjon osv..

Argument inneholder en array med verdier når funkjsoner blir kalt, function (2,53,6,35,7,1,44,63,64); Man kan bruke det til å gå gjennom hele arrayen for å legge det sammen, finne gjennomsnitt, største tall, minste tall. osv.

Parameter er kort sagt det ukjente i et liknings regnestykke, men i funksjoner legger vi verdiene til når vi kaller på funkjonen.

function(tall1, tall2) {
var sum = tall1 +tall2;
return svar;
}
